Project 3 - Student Loan Calculator

Student Loan Calculator

This simple loan calculator utility helps students plan for their loan returns by calculating and showing the amount payable which will be due in amortization period.

One of the best feature of this tool is its portability. The tool is completely developed in java which is a operating system independent language. So doesn't matter if students want to use it on their Windows Laptop or their Mac OSX PC or on a Ubuntu Device, the tool will work seamlessly on all devices.
