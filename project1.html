Project 1 - Book Cipher Encoder

BCEncode Program

BCEncode is a Encryption Utility that can be used for encrypting any text. The program requires the user to enter the path of the book file which will be used as a key for encryption and path the file containing secret message. The program also asks user to enter a third parameter which is a path to the output file in which it'll store the encrypted text.

One great thing about this utility is that it can encrypt not only text but can also encrypt any file at all. This application is developed to work at binary level. So it is capable enough to work on the bytes level.
