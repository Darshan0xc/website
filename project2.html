Project 2 - File Usage Utility

Fileusage File Utility

Fileusage is a simple minified tool similar as 'ls' utility in linux and 'dir' in windows. But unlike those tools, this tool allows end user to use different filters and categorization to sort and view files in a friendly clear grid view.

Fileusage by default supports files of C, C++, Java, Web Files etc. But the program also supports regular expression which allows users to create their own filter using regex patterns.
